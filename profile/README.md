# NimiqMiniApp.org

NimiqMiniApp.org is a community-driven open source organization dedicated to making it easy to build, learn, and share Nimiq Mini Apps.

Our goal is to grow the ecosystem by providing high-quality developer resources, reusable libraries, example applications, and production-ready templates.

What you'll find here
📦 Mini App SDK utilities
🚀 Starter templates
🎮 Example Mini Apps
📚 Documentation and guides
🧩 UI components
🛠 Development tools
💡 Community experiments
🔌 APIs and integrations
Our mission

We want anyone—from complete beginners to experienced developers—to be able to build amazing Mini Apps for the Nimiq ecosystem with as little friction as possible.

By keeping everything open source, we hope to encourage collaboration, learning, and innovation throughout the ecosystem.

Contributing

Contributions are always welcome!

Whether you're fixing a typo, building a new Mini App, improving documentation, or creating a new library, we'd love your help.
